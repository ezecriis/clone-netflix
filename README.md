# Liteflix

Netflix clone

## ⚙️ Run Web app locally

Make sure that you have the [Yarn](https://yarnpkg.com/en/) or `npm` installed, then install dependecies:

```sh
yarn install
```

or

```sh
npm install 
```

to finally run the app

```sh
npm run develop
```
